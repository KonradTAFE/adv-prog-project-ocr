
**Tab Navigation:**

- Tab cycles through all interactive elements in logical order:
    - Recent dropdown → Settings → Open File → URL field → Load URL → Slider → Skip buttons → Playback controls → Speed → Capture

**Keyboard Shortcuts:**

- **Space** or **K** - Toggle play/pause
- **S** - Stop
- **Left/Right arrows** - Skip short amount (backward/forward)
- **J/L** - Skip long amount (backward/forward) - YouTube style
- **Shift+,** / **Shift+.** - Decrease/increase playback speed
- **C** - Capture frame
- **Ctrl+O** - Open file
- **Ctrl+,** - Open settings (Mac standard)
